Сетевая часть проекта - Netty.

И клиент, и сервер - консольные приложения.

Для обмена информацией исспользуются специальные классы обертки (в пакете Messages), проходящие через ObjectDecoder/Encoder. Это накладывает ограничение на размер пересылаемого файла. В дальнейшем попытаюсь уйти от этого, с помощью интерфейса ChunkedInput.

База данных пользователей - MySQL. На данный момент реализованна минимальная необходимая функциональность: регистрация пользователя, логин уже зарегистрированных пользователей.

Работа с файлами и фс. На данный момент релизовано создание ндивидуальной директории для каждого пользователя на серверной стороне, а так же отправка файла с клиента на сервер с помощью комнды /upload. В окончательном варианте бужет реализованна возможность просмотреть список файлов на сервере и закачка файлов на клиент.
